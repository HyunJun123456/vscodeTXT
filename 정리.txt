사람 -> 명령 (Shell) -> OS -> 제어 -> 하드웨어 (CPU, RAM)

공개키 기반 암호화 방식
데이터 암호화: 상대방의 공개키로 잠근다
데이터 전송 방법
1. 데이터를 수신자의 공개키로 잠근다. -> 암호화
2. 그 데이터를 다시 송신자의 비밀키로 잠근다. -> 전자서명
3. 전송한다.
데이터 받는 방법
1. 데이터를 송신자의 공개키로 연다. -> 서명 검증
2. 열리면 수신자의 개인키로 연다. -> 복호화

AWS에 22번에 접속할 때 비밀키로 접근하면
AWS에 .ssh 폴더에 Authorized_key에 공개키가 존재해서
누가 접속하는지에 대해 구분이 가능함
세션이 생성되면 그 뒤로부터는 무제한 통신 가능함
