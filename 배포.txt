1. 자바 설치 (JDK, JRE)
sudo apt update
sudo apt install openjdk-11-jdk
- JDK로 마인크래프트 게임 개발
- 일반 사용자들이 마인크래프트 게임 하려면 JRE가 필요!!

2. 깃허브 다운받기
git clone 주소

3. gradlew 실행권한주기
cd ~/aws-v1
chmod u+x gradlew

4. gradlew로 프로젝트를 jar파일로 변경하기
./gradlew build
./ -> 현재폴더

5. java로 jar를 실행
cd build/libs
java -jar *.jar
